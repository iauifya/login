# login/logout side project

### 使用技術
* vue3開發 + vue cli環境部屬
  * 使用set up語法糖開發 
* vuetify，Based on Vue的CSS Framework
* vee-validate 表單驗證套件搭配撰寫前端驗證機制 
* swiper 幻燈片套件
* firebase部屬網站
